# encrypter
Overview  This is a basic Python script that implements a simple substitution cipher. It converts each letter (A–Z, case-insensitive) into a corresponding symbol, while leaving non-alphabet characters unchanged.

Usage
1. Run the script
encryption.py
2. Enter text when prompted
What do you want to encrypt: Hello World
3. Output example
+^~~{ -{]~%

📦 Requirements
Python 3.x
No external libraries required
